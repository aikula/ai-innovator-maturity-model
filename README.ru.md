# AI-Innovator Maturity Model v0.1

**Двуязычная модель зрелости для ИИ-систем, которые выходят за пределы выполнения задач и переходят к рыночным, продуктовым и бизнес-модельным инновациям.**

**Автор:** Андрей Кулинич / Kulinich.AI  
**Версия:** v0.1 public draft  
**Дата:** 2026-06-17  
**DOI:** [10.5281/zenodo.20737884](https://doi.org/10.5281/zenodo.20737884)  
**Языки:** английский и русский  
**Статус:** концептуальная рамка, не бенчмарк и не заявление об AGI

## Зачем этот репозиторий

ИИ-агенты становятся стандартным интерфейсом выполнения работы. Следующий рубеж — не только системы, которые действуют, а системы, которые **исследуют пространство новых возможностей**: новые продукты, бизнес-модели, категории рынка, ценовые стратегии, рекламные механики и операционные модели.

Этот репозиторий предлагает **AI-Innovator Maturity Model v0.1**: практическую классификацию, которая отделяет простой генератор идей от серьёзной ИИ-системы, способной:

- собирать доказательства;
- формулировать проверяемые гипотезы;
- сочетать строгие финансово-операционные модели с поведенческими симуляциями;
- запускать вероятностные сценарии;
- проектировать реальные эксперименты;
- учиться на расхождении прогноза и факта;
- искать подрывные изменения бизнес-модели.

Главный тезис:

> AI-Innovator становится game changer только тогда, когда замыкает цикл: идея → данные → модель → симуляция → эксперимент → факт → обучение.

Без этого цикла это просто чат-бот в костюме стратега. А таких человечество уже наплодило с каким-то особенно унылым энтузиазмом.

## Краткая версия модели

| Уровень | Название | Суть |
|---|---|---|
| M0 | Idea Generator | Генерирует идеи без доказательств и проверки |
| M1 | Market Research Assistant | Собирает и структурирует рыночные данные |
| M2 | Hypothesis Designer | Превращает идеи в проверяемые гипотезы |
| M3 | Simulation-Based Innovator | Проверяет гипотезы через строгие операционные и финансовые модели |
| M4 | Probabilistic Market Simulator | Запускает множество сценариев и показывает распределения, а не один прогноз |
| M5 | Self-Learning Market Lab | Обновляет модель на основе реальных экспериментов |
| M6 | Business Model Hacker | Ищет новые категории, единицы ценности и правила конкуренции |

## Почему сначала крупный бизнес

Первые практические покупатели — банки, телеком, крупный ритейл, страховые компании, промышленность, энергетика, авиакомпании, медицинские сети, крупная недвижимость и wellness-операторы.

У них есть три вещи, которых часто нет у малого бизнеса:

1. **Данные**: транзакции, CRM, канальная аналитика, операции, поведение клиентов, расходы.
2. **Острая конкуренция**: даже небольшой сдвиг в модели может двигать большие деньги.
3. **Возможность экспериментов**: пилоты, A/B-тесты, региональные запуски, ограниченные предложения, операционные симуляции.

Для них AI-Innovator — не игрушка для брейншторма, а способ снизить стоимость стратегических ошибок и найти новые рыночные ходы до того, как в них вложены большие бюджеты.
